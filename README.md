# todo-vue
- 프로젝트전 Vue3 공부

### json server 사용
- npm install -g json-server
- json-server --watch db.json
- [json-server](https://github.com/typicode/json-server)

### 구현된 기능
- to-do 추가
- to-do 완료 체크
- to-do 삭제
- to-do 수정 (데이터 변경시에만 수정 버튼 활성화)
- to-do 읽기
- to-do 검색
- 페이지네이션
- 토스트(데이터 추가 or 업데이트 하단 알림)
